## Introduction

Cafe is a basic ecommerce website based on Laravel 5.7 and Vuejs

## Table of Contents

1. [Requirements](#requirements)
2. [Installation](#installation)
3. [Features](#features)
4. [Images][#images]

## Requirements

SEVER REQUIREMENTS:

- PHP >= 7.1.3
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension
- Ctype PHP Extension
- JSON PHP Extension
- BCMath PHP Extension

OPTIONAL REQUIREMENTS:
- Composer
- NPM / YARN

## Installation

1. `git clone https://github.com/loger-projects/cafe.git`
2. `cd cafe`
3. `composer install`
4. `php artisan migrate`
( optional )
5. `php artisan db:seed`

## Features

- Responsive layout
- Lazyload images
- Admin Dashboard
- Book Table
- Shopping Cart
- Restaurant Menu
- Blog
- Comment (3 levels)
- Review(1 level - rating)
- Media
- User Login - Register - Email Verify - Forgot Password
- User Dashboard ( manage orders, favorite food - drink, review, comment)
